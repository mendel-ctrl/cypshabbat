# CYP Shabbat — Happy Hour, Holy Hour

A single-page event website for **CYP Shabbat**, hosted by Chabad Young Professionals of St. Petersburg.

**Event:** Friday, August 7, 2026 · 7:00 PM Happy Hour · 7:45 PM Shabbat Dinner
**Location:** Chabad Center for Jewish Life, 533 4th Street North, St. Petersburg, FL

## Files

- `index.html` — the complete website (HTML, CSS, and JavaScript in one file)
- `images/` — event photography used across the page

## Editing

- **Registration link:** in `index.html`, search for `--rsvp-url` and change the address in quotes. That one line updates every Reserve/RSVP button.
- **Photos:** replace the files in the `images/` folder (keep the same file names), or point the `src` / `background-image` values to new files.
- **Colors:** search for `DESIGN TOKENS` near the top of `index.html`; each color is a named line you can change.
- **Wording & times:** the text lives directly in the page — search for what you want to change and type over it.

## Viewing

Open `index.html` in any web browser, or publish the folder with GitHub Pages
(Settings → Pages → deploy from the `main` branch) for a live link.
